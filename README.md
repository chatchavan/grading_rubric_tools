# Tools for desiging and applying grading rubric

[Live demo](https://chatw.ch/grading_rubric_tools/)

Both tools run locally on your web browser. If you like, you can download [the source code](https://github.com/chatchavan/grading_rubric_tools) and double-click on the HTML files. In that way, everything is self-contained on your computer.
